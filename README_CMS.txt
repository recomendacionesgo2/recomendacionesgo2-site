Go2 CMS (Decap) – Setup rápido

Archivos incluidos:
- admin/config.yml : configuración del CMS (colección Productos).
- images/uploads/.keep : carpeta destino de imágenes.
- content/products/.keep : carpeta de ficheros de producto.

Cómo usar
1) Sube todo el contenido de este ZIP a la raíz del repo recomendacionesgo2-site.
2) En Netlify → Deploys, pulsa Trigger deploy → Deploy site.
3) Abre https://recomendacionesgo2.netlify.app/admin e inicia sesión.
4) New → Productos para crear. Marca “Destacado” para el Producto destacado.

Recomendación de imágenes: cuadradas 800×800 o superiores.
